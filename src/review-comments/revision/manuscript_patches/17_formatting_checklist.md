# Patch 17 — IJIES Formatting Checklist

Authors must walk through this checklist on the *final* Word document before resubmission. Each item is verbatim from the editor's note in `comment-264 INASS Revision Comment.docx`.

## Body and tables

- [ ] Body sentences set to **11 pt**.
- [ ] All tables set to **10 pt**.
- [ ] Equations set to **11 pt**, italic font for mathematical expressions.
- [ ] Equation environment is *plain text* — no Microsoft Math / OMML environment in inline sentences.
- [ ] No `*` used in equations (replace with `\cdot` or `\times`).
- [ ] Equations indented 5 mm; one blank line above and below each equation.
- [ ] Equation numbers right-aligned and continuous.

## Figures

- [ ] All figures redrawn at ≥ 300 DPI.
- [ ] Letter size in figures unified to **≥ 10 pt**.
- [ ] Wide figures (Figs 1, 3, 5, 6, 7, 9, 10) set in **one-column** width spanning the page.
- [ ] No two figures placed side-by-side in one column.
- [ ] No `(a)`, `(b)`, `(c)` labels embedded inside figure images — use caption text instead.
- [ ] No borders around the outside of figure boxes.
- [ ] No figure split across two pages.
- [ ] No figures used as substitutes for tables or equations.
- [ ] All figures original to this work (no third-party copyrighted images).

## Layout

- [ ] No rotated sheets or vertical writing.
- [ ] Figures and tables placed at the top or bottom of each column.
- [ ] Figures referenced as **Fig. 1, Fig. 2, …**; tables as **Table 1, Table 2, …**.
- [ ] Equations referenced as **Eq. (1), Eq. (2), …**.

## Author and affiliation block

- [ ] No titles (Dr., Prof., etc.) in author or affiliation names.
- [ ] No acronyms or abbreviations in author or affiliation names.
- [ ] No repeated affiliations — combine identical institutions into a single entry.

## Citations

- [ ] All references cited as `[1]`, `[2-4]` style.
- [ ] First-name initials used (e.g. `R. Ruskone`, not `Ruskone, R.` or `Robert Ruskone`).
- [ ] Journal references include **Vol., No., pp., year, doi** in IJIES sample format.
- [ ] Conference references include **In: Proc. of …, location, pp., year, doi** in IJIES sample format.
- [ ] Reference list re-checked for missing volume / issue / page fields.

## Length

- [ ] Manuscript ≥ **8 pages** in IJIES one-column template (with the new §3.10, §4.3, §5.4, §5.5, §5.6, Appendix A this is comfortably satisfied).

## Track-changes hygiene

- [ ] Every *added or modified* sentence in this revision is in **red font**.
- [ ] No remaining placeholder strings such as `« fill »`, `« N_FINAL »`, `«N_MOD»` etc. — search the document for `«` to verify.
- [ ] Cross-references to new sections (§3.10, §4.3, §5.4, §5.5, §5.6, Appendix A) renumber correctly throughout the manuscript.

## Final export checks
